# Calibration Patterns

The three onboarding questions are in [RULES.md](../RULES.md). This directory collects deeper patterns for catching subtler boxing after the first layer lands.

## Contributing

Found a question or pattern that surfaced a correction you didn't expect? Add it. A good calibration question:
- Targets one behavioral dimension
- Produces answers that pass the golden rule (changes AI behavior)
- Exposes AI assumptions rather than collecting biography
